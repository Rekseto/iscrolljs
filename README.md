# iscrolljs
Smooth scroll **without jQuery**, that pretty library weights only **4.65kb**.
## Dependencies
*iscrolljs** is usign https://github.com/iamdustan/smoothscroll as polyfill to smooth scroll behavior.


## Sample usage
Can be found in /test directory ;)

## Contribution
### Install packages
```
npm install
```
### Gulp CLI
```
npm install -g gulp
```
### Build
```
npm run build
```
or
```
gulp build
```
